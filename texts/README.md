# Texts

This is a sketch. The goal is to try to find a simple but viable format to archive every work on the human condition, and all translations and commentaries thereof.

The minimal possible directory structure & markdown metadata structure is best. The point isn't to ontologize human knowledge, but to assemble it in a form that can be read and cross referenced by AI, and humans using AI, to better understand ourselves.

## Texts
